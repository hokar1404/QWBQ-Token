Purpose of This Disclosure

This page clearly defines and publishes the only official email communication channels associated with the Quazenthor project.

It exists to:

  • Provide transparent contact information for technical and documentation-related matters
  • Enable clear identification of official project correspondence
  • Help prevent impersonation and unauthorized use of the project name
  • Support consistent and verifiable communication aligned with the project's educational purpose

The Quazenthor token (QWBQ) is a minimal, immutable BEP-20 reference implementation deployed on BNB Smart Chain mainnet for real on-chain educational interaction.

Official Project Email

The single official email address representing the Quazenthor project is:

admin@quazenthor.com

This address is managed by the project founder and is used exclusively for:

  • Technical clarifications about the smart contract and its behavior
  • Documentation and transparency-related correspondence
  • Coordination with contributors on educational materials

This email is not used for:

  • General public inquiries or support requests
  • Any form of financial, investment or trading discussion
  • Promotional or marketing purposes
  • Token distribution requests outside documented educational channels

Only emails sent from the @quazenthor.com domain are official. Any other domain or address claiming association should be considered unauthorized.

Impersonation Prevention Guidelines

To verify authenticity of any communication claiming to be from Quazenthor:

  • Check that the email comes from an @quazenthor.com address
  • Cross-reference with official profiles listed on the Network Links & Team page
  • Verify contract ownership and signed messages directly on BscScan
  • Ignore any unsolicited messages not linked to this official page or verified channels

Communication Scope

All official communication is strictly limited to:

  • Technical questions about the BEP-20 contract implementation
  • Clarification of published documentation
  • Educational use-cases of the reference deployment

The project maintains a strict non-financial policy. No communication channel is used for, or accepts discussion of:

  • Token value, trading or market activity
  • Investment, profit expectation or financial opportunity
  • Any form of commercial or promotional activity

Final Statement

Quazenthor (QWBQ) is a transparent, educational mainnet reference deployment. All official communication follows the channels and limitations described above.

The project website remains the authoritative source of information. Updates are tracked in changelog.html.
